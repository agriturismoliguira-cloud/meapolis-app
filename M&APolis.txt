Valutate opzioni per convertire in app scaricabile
Valutate opzioni per convertire in app scaricabile
Hai due strade principali:

1. PWA — Progressive Web App (consigliata per iniziare) La più veloce e economica. Si aggiunge alla home screen come un'app vera, funziona offline, non richiede App Store. Lo sviluppatore aggiunge un manifest.json e un service worker al sito esistente — è un giorno di lavoro su quello che abbiamo già fatto.

Limiti: non appare su App Store/Google Play, alcune funzioni native (notifiche push) sono limitate su iOS.

2. App nativa con Expo / React Native Il codice React che abbiamo fatto è molto vicino a React Native — uno sviluppatore può portarlo con modifiche moderate. L'app finisce su App Store e Google Play.

Costi aggiuntivi: account Apple Developer (€99/anno), account Google Play (€25 una tantum), più tempo di sviluppo.

Il mio consiglio per Li Guira:

Parti dalla PWA. I tuoi ospiti DACH e olandesi ricevono il link via WhatsApp o email prima del check-in, lo salvano sulla home screen, e hanno tutte le info offline anche senza connessione in zona Stio. È esattamente quello che serve — e costa pochissimo.

L'app nativa ha senso solo se in futuro vuoi integrare notifiche push (es. "il fuoco è acceso stasera") o vuoi presenza sugli store come canale di brand.

Vuoi che preparo il manifest.json e le istruzioni PWA per lo sviluppatore?

Identificato quesito Git generico sul progetto meapolis
Il progetto meapolis è un singolo file HTML, quindi è semplicissimo. Ecco i passi:

Identificato quesito introduttivo su avvio Git
Git non si "avvia" come un'app — è uno strumento da terminale. Ecco come usarlo:

git --version
Se vedi un numero tipo git version 2.43.0 sei a posto. Se dice "comando non trovato" devi installarlo da git-scm.com.

3. Configuralo la prima volta (una volta sola)

git config --global user.name "Tuo Nome"
git config --global user.email "tua@email.com"

Visto che hai un solo file HTML e vuoi caricarlo su GitHub, il metodo più veloce senza toccare il terminale è quello che ti ho detto prima: vai su github.com, crei il repository e trascini il file direttamente nel browser.

/* ══════════════════════════════════════════════════════════════
   NAVIGAZIONE

function goTo(page) {
  document.querySelectorAll('.page').forEach(p => p.classList.remove('active'));
  document.getElementById('page-' + page).classList.add('active');
  document.querySelectorAll('.nav-btn').forEach(b => {
    b.classList.toggle('active', b.dataset.page === page);
  });
  window.scrollTo(0, 0);
  if (page === 'visite') chiudiVisita();
}
 
/* ══════════════════════════════════════════════════════════════
   RENDER VISITE
══════════════════════════════════════════════════════════════ */
function renderVisite() {
  const container = document.getElementById('visite-list');
  container.innerHTML = VISITE.map(v => `
    <button class="visita-card" onclick="apriVisita('${v.id}')">
      <p class="v-data">${v.data}${v.prossima ? ' · <span style="color:#9B2335">Prossima</span>' : ''}</p>
      <h3>${v.nome}</h3>
      <p>${v.desc.substring(0, 90)}…</p>
      <span class="v-arrow">→</span>
    </button>
  `).join('');
}
 
function apriVisita(id) {
  const v = VISITE.find(x => x.id === id);
  if (!v) return;
 
  document.getElementById('det-data').textContent = v.data;
  document.getElementById('det-nome').textContent = v.nome;
  document.getElementById('det-desc').textContent = v.desc;
 
  const luoghiHTML = v.luoghi.map(l => `
    <div class="det-luogo">
      <span class="luogo-icon">${l.icon}</span>
      <p>${l.nome}<span>${l.nota}</span></p>
    </div>
  `).join('');
  document.getElementById('det-luoghi').innerHTML =
    `<h4>Luoghi visitati</h4>${luoghiHTML}`;
 
  document.getElementById('visite-list-wrapper').classList.add('hidden');
  document.getElementById('visita-detail').classList.add('active');
  window.scrollTo(0, 0);
}
 
function chiudiVisita() {
  document.getElementById('visite-list-wrapper').classList.remove('hidden');
  document.getElementById('visita-detail').classList.remove('active');
  window.scrollTo(0, 0);
}
